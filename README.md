# Jokempo
Projeto em C# que implementa o jogo Jokenpô (Pedra, Papel e Tesoura) em console. O sistema inclui modularização com métodos, validação de entrada de dados, registro e troca de jogadores, além de exibição de estatísticas como vitórias, derrotas e empates.

Tecnologias utilizadas

C#

.NET / Console Application

Git e GitHub para versionamento do código.
